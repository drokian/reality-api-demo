# ETHICS

Reality API Demo communicates scientific and philosophical ideas that can be easily overinterpreted.
This document defines communication boundaries.

## Ethical Commitments

1. Intellectual honesty
   - We separate verified observations from speculative interpretation.
2. Source traceability
   - Key claims should be backed by references when possible.
3. Uncertainty transparency
   - Unknowns and open questions are explicitly stated.
4. Public responsibility
   - Language should not encourage pseudoscientific certainty.
5. Respectful discourse
   - Criticism and alternatives are welcome when evidence-based.

## Communication Guardrails

- Do not present hypotheses as established fact.
- Do not frame uncertainty as weakness.
- Do not hide contradictory or incomplete evidence.

## Data Ethics

- Demo data is a curated public snapshot.
- No personal or sensitive information should be published.
- Provenance and update timing should remain clear.

## AI Usage Note

AI-assisted text or analysis is support material, not a substitute for scientific validation.
Human review remains mandatory.
