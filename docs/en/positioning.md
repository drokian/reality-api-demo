# Positioning

## What Reality API Demo Is

Reality API Demo is a public research showcase.
It presents observations, hypotheses, analyses, and references from the broader Reality API effort.

## What It Is Not

- Not a full research operations system
- Not a final scientific theory
- Not a simulation proof claim
- Not a replacement for peer-reviewed validation

## Difference from Similar Narratives

### Versus generic simulation-theory content

- Focuses on structured records instead of pure discussion
- Keeps observation and reference linkage explicit
- Prioritizes falsifiable framing

### Versus closed internal research notebooks

- Public and inspectable output
- Bilingual public communication
- Static snapshots for consistency and auditability

### Versus production platform

- Demo: read-only and static
- Full platform: database-backed, admin-enabled, operational

## Audience

- Curious public readers
- Technical reviewers
- Potential collaborators
- Media and outreach channels

## Outcome Goal

Create high-trust first contact for the project: clear idea, clear limits, clear evidence path.
